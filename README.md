# .github

Repositorio de gobernanza común de la organización **Libertad VZLA**.

GitHub le da un trato especial a un repositorio llamado `.github`: lo que vive acá se aplica a toda la organización.

| Archivo | Para qué sirve |
|:---|:---|
| [`profile/README.md`](./profile/README.md) | Es la portada pública de la organización (lo que se ve en github.com/Libertad-Vzla). |
| [`CODE_OF_CONDUCT.md`](./CODE_OF_CONDUCT.md) | Código de conducta para todos los espacios de la organización. |
| [`CONTRIBUTING.md`](./CONTRIBUTING.md) | Guía de cómo colaborar. |
| [`SECURITY.md`](./SECURITY.md) | Cómo reportar una vulnerabilidad de seguridad. |
| [`LICENSE`](./LICENSE) | Licencia de la documentación (CC BY 4.0). |

Estos archivos funcionan como valores por defecto: cualquier repositorio público de la organización que no tenga su propia versión hereda la de acá. Así no hay copias repetidas.

Para conocer el proyecto en detalle, ver [libertad-showcase](https://github.com/Libertad-Vzla/libertad-showcase). El sitio público está en [libertadvzla.com](https://www.libertadvzla.com).
