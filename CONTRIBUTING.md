# Cómo colaborar con Libertad VZLA

El código fuente de la plataforma es **privado por política de seguridad operacional**. Las contribuciones públicas se reciben en estas formas:

- ✏️ **Correcciones editoriales** → [política de correcciones](https://www.libertadvzla.com/about/corrections).
- 📨 **Reportes ciudadanos** → [formulario `/reportar`](https://www.libertadvzla.com/reportar).
- 🛡️ **Vulnerabilidades de seguridad** → ver [`SECURITY.md`](./SECURITY.md).
- 💬 **Propuestas editoriales o de investigación** → `contacto@libertadvzla.com` con propuesta y antecedentes.
- 📰 **Solicitudes de prensa** → `contacto@libertadvzla.com`.

## Código de conducta

La participación en cualquier espacio de esta organización está regida por nuestro [Código de conducta](./CODE_OF_CONDUCT.md).

## English

The platform source code is private by security policy. Public contributions are received via the channels above: editorial corrections, citizen reports, security disclosures, partnership proposals, and press inquiries. All participation is governed by our Code of Conduct.
